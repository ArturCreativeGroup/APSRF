# What Is APSRF?

APSRF (Agentic Protocol Security Research Framework) is a research framework designed to study protocol behavior, governance, security risks and emergent effects within AI agent ecosystems.

The framework provides a structured methodology for analyzing:

* prompts,
* protocols,
* workflows,
* agents,
* orchestration systems.

without requiring access to model internals.

---

# Why APSRF Exists

Modern AI systems are increasingly controlled by protocol ecosystems rather than by models alone.

Examples include:

* custom GPT instructions,
* prompt frameworks,
* agent chains,
* workflow automation,
* orchestration layers.

These systems introduce new governance and security challenges that traditional cybersecurity tools are not designed to evaluate.

APSRF was created to investigate and address these challenges.
